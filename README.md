# TokenBarPublic

TokenBar is a macOS menu bar app that shows live AI token usage so you can control costs before they get out of hand.

## What It Does

- Tracks token usage in real time
- Helps catch unusual usage spikes early
- Makes usage and spend easier to understand

## How It Works

1. Install and open TokenBar.
2. Keep it running in your menu bar.
3. Use your normal AI workflow.
4. TokenBar updates usage live while you work.
5. Check totals to spot waste and optimize prompts/models.

## Install (macOS)

1. Download the latest `TokenBar.dmg` from the Releases page.
2. Open the DMG file.
3. Drag **TokenBar** into **Applications**.
4. Launch TokenBar from **Applications**.
5. If macOS warns on first launch, right-click the app and choose **Open** once.

## Quick Usage

1. Open TokenBar.
2. Start your AI work.
3. Watch live token counts from the menu bar.
4. Use the numbers to decide when to pause, adjust prompts, or switch models.
